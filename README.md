# UI-UX-Experiment3

## Aim:
To perform a heuristic evaluation of two leading e-commerce websites — Amazon and Flipkart — based on Nielsen’s 10 Usability Heuristics, and to develop and incorporate design improvements to enhance their overall user experience (UX).

## Algorithm:
1. Selection of Platforms: Choose two similar platforms (Amazon and Flipkart) offering comparable services for fair UX evaluation.

2. Define Evaluation Criteria: Apply Nielsen’s 10 Usability Heuristics as the evaluation framework.

3. Conduct Heuristic Evaluation:

4. Evaluate each heuristic on both websites.

5. Identify usability issues and assign severity ratings (1 = minor, 4 = critical).

6. Record observations in a comparison table.

7. Analyze Findings: Compare how each website performs under each heuristic and identify overlapping weaknesses.

8. Develop Design Improvements:

9. Propose UI/UX solutions for high-severity issues.

10. Create prototype concepts (Undo Snackbar, Inline Coupon Validation, Sticky Filter Bar, etc.).

11. Implement and Incorporate Changes: Simulate solutions using code snippets and mock layouts to show how the changes improve usability.

12. Evaluate Outcomes: Assess how the proposed solutions enhance control, feedback, and efficiency for users.
